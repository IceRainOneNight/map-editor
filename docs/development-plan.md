# 开发计划

## 总览

本项目分 5 个 Phase 逐步推进，每个 Phase 独立可交付、可验证。

**原则**：
- 每完成一个 Phase，项目都是可运行的
- Phase 内部任务粒度适中，不过大不过小
- 遇到问题优先在 Phase 内解决，不跨 Phase 埋坑

---

## Phase 1：项目脚手架 & 底图集成

**目标**：搭建 React 项目，集成 MapLibre GL JS，渲染必应底图，可切换底图。

### 任务清单

| # | 任务 | 预计产出 |
|---|------|---------|
| 1.1 | Vite + React + TypeScript 项目初始化 | 可 `npm run dev` 启动 |
| 1.2 | 安装核心依赖：maplibre-gl, zustand, Tailwind CSS | package.json 就绪 |
| 1.3 | 创建基础目录结构和类型定义 | src/ 结构建立 |
| 1.4 | 实现 MapView 组件（MapLibre 初始化） | 地图渲染出来 |
| 1.5 | 配置必应地图底图源（默认）+ 高德底图源 | 两种底图 URL 配置 |
| 1.6 | 实现 BasemapSwitcher 底图切换组件 | 可切换底图 |
| 1.7 | 实现 mapStore（Zustand）管理底图状态 | 状态持久化 |
| 1.8 | 添加基础 MapControls（缩放按钮） | 地图控件可用 |

### 验收标准
- [ ] 页面加载后显示必应地图底图
- [ ] 可切换为高德底图，切换流畅
- [ ] 滚轮缩放、拖拽平移正常
- [ ] 项目可 `npm run build` 生产构建

## Phase 2：数据加载 & 图层管理

**目标**：加载 GeoJSON/SHP 文件，在地图上渲染，管理多个图层。

### 任务清单

| # | 任务 | 预计产出 |
|---|------|---------|
| 2.1 | 实现 FileMenu 文件加载组件（拖拽+点击） | 文件选择器 UI |
| 2.2 | 实现 GeoJSON 解析和 MapLibre source/layer 渲染 | GeoJSON 显示在地图上 |
| 2.3 | 集成 shpjs 实现 SHP 解析 | SHP 文件可加载渲染 |
| 2.4 | 实现坐标系检测和转换（proj4） | WGS84/GCJ-02/BD-09 互转 |
| 2.5 | 实现 LayerPanel 左侧图层面板 UI | 图层列表显示 |
| 2.6 | 实现图层可见性开关、排序、删除、激活 | 图层管理交互 |
| 2.7 | 实现 layerStore（Zustand）管理多图层状态 | 图层状态管理 |
| 2.8 | 图层加载时的样式自动分配（不同颜色） | 多图层可区分 |

### 验收标准
- [ ] 可加载 .geojson / .json 文件并渲染
- [ ] 可加载 .shp 文件并渲染
- [ ] GCJ-02 和 BD-09 数据正确叠加在高德底图上
- [ ] 图层面板可管理多个图层
- [ ] 图层可见性和顺序可控

## Phase 3：编辑功能

**目标**：实现地图要素的绘制、修改、删除和属性编辑，完整的 Undo/Redo。

### 任务清单

| # | 任务 | 预计产出 |
|---|------|---------|
| 3.1 | 实现 DrawToolbar 绘制工具 UI（选择/点/线/面） | 工具栏绘制按钮组 |
| 3.2 | 集成 MapLibre Draw 或自行实现点/线/面绘制 | 可在地图上绘制要素 |
| 3.3 | 实现要素选择和节点编辑（拖拽修改形状） | 选中高亮 + 节点拖拽 |
| 3.4 | 实现要素删除 | Delete 键/按钮删除 |
| 3.5 | 实现 AttributePanel 属性面板 UI | 右侧属性面板 |
| 3.6 | 实现属性编辑并同步到图层数据 | 编辑属性后数据更新 |
| 3.7 | 实现 editStore + zundo Undo/Redo 系统 | Ctrl+Z 撤销 |
| 3.8 | 绘制/编辑完成后的数据写回 layerStore | 数据流闭合 |

### 验收标准
- [ ] 可绘制点、线、面
- [ ] 可选中要素并拖拽节点修改
- [ ] 右面板显示并可编辑属性
- [ ] Undo/Redo 可用
- [ ] 删除要素正常

## Phase 4：导出功能

**目标**：实现图片截取、GeoJSON 导出、Shapefile 导出。

### 任务清单

| # | 任务 | 预计产出 |
|---|------|---------|
| 4.1 | 实现地图视口截图（MapLibre getCanvas + 导出 PNG） | 截图下载 |

| 4.3 | 集成 shp-write 实现 Shapefile 导出 | SHP 生成 |
| 4.4 | 使用 JSZip 打包 SHP 导出文件 | .zip 下载（含 shp/dbf/prj） |
| 4.5 | 导出时坐标系选择 UI | 导出对话框 |
| 4.6 | 实现 ExportMenu 导出菜单组件 | 统一的导出入口 |

### 验收标准
- [ ] 可截图当前视口保存为 PNG
- [ ] 可导出 GeoJSON
- [ ] 可导出 Shapefile (.zip)
- [ ] 导出可选目标坐标系
