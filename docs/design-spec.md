# 设计规范

## 设计参考：QGIS 风格

参考 QGIS 的桌面 GIS 编辑器布局，适配 Web 环境：
- 三栏布局：左图层面板 + 中地图 + 右属性面板
- 顶部工具栏，底部状态栏
- 深灰色主题为主，保留专业感

---

## 布局规格

### 整体布局

```
┌──────────────────────────────────────────────────────────┐
│  Toolbar (48px)                                          │
│  [文件▼] [底图▼] [绘制·□◯] [撤销↩] [重做↪] [✕删除] [📷截图] │
├──────────┬────────────────────────┬──────────────────────┤
│          │                        │                      │
│  Layers  │       Map View         │    Attribute Panel   │
│  Panel   │                        │                      │
│  280px   │      flex: 1           │       320px          │
│          │                        │                      │
├──────────┴────────────────────────┴──────────────────────┤
│  Status Bar (28px)                                       │
│  🟢 图层: 3  |  坐标系: WGS84  |  鼠标: 116.397, 39.908 │
└──────────────────────────────────────────────────────────┘
```

### 尺寸规范

| 区域 | 宽度/高度 | 说明 |
|------|-----------|------|
| 工具栏 | 100% × 48px | 固定高度 |
| 图层面板 | 280px | 可折叠 |
| 地图区 | flex: 1 | 自适应 |
| 属性面板 | 320px | 可折叠/收起 |
| 状态栏 | 100% × 28px | 固定高度 |
| 最小分辨率 | 1280 × 720 | |

## 色彩方案

### 主题色

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主背景 | 深灰 | #2D2D2D |
| 面板背景 | 中深灰 | #383838 |
| 面板边框 | 深色线 | #4A4A4A |
| 主文字 | 浅灰白 | #E0E0E0 |
| 次要文字 | 灰色 | #999999 |
| 强调色 | QGIS 黄绿 | #8CB82A |
| 选中态 | 高亮蓝 | #4A90D9 |
| 错误/删除 | 红色 | #E74C3C |
| 悬停态 | 微亮 | rgba(255,255,255,0.05) |

### 底图相关

- 图层要素默认颜色：`#3388ff`（蓝色点/线/面）
- 选中要素颜色：`#FFD700`（金色）
- 编辑节点颜色：`#FF4500`（橙红）
- 填充透明度：`0.2`

## 组件规范

### 工具栏 (Toolbar)

```
高度: 48px
背景: #2D2D2D
布局: flex, 左对齐
间距: 按钮间 4px, 分组间 16px

按钮:
  - 高度: 36px
  - 内边距: 8px 12px
  - 图标 + 文字 或 仅图标
  - hover: 背景 rgba(255,255,255,0.08)
  - active: 背景 rgba(255,255,255,0.12) + 底部 2px #8CB82A 指示线

分组:
  FileMenu | BasemapSwitcher | separator | DrawTools | EditControls | separator | Export
```

### 图层面板 (LayerPanel)

```
宽度: 280px
背景: #383838
标题栏: 36px, "图层", 左侧
图层列表:
  - 每项高度: 40px
  - 拖拽手柄 (左 4px) + 可见性复选框 + 图层名 + 更多菜单
  - 激活图层: 左侧 3px #8CB82A 指示条
  - 悬停: 背景 rgba(255,255,255,0.04)
  - 选中: 背景 rgba(255,255,255,0.08)
底部: 添加图层按钮
```

### 属性面板 (AttributePanel)

```
宽度: 320px
背景: #383838
无选中时: 显示 "点击地图上的要素查看属性"
有选中时:
  - 标题栏: 要素类型图标 + 要素 ID
  - 属性列表: 键值对表单
  - 每行: label + input/textarea
  - 只读字段灰色显示
  - 编辑后自动保存到图层数据
```

### 地图控件 (MapControls)

```
右上角:
  - 缩放按钮组 (+ / -)
  - 间距: 2px
  - 尺寸: 32×32px
  - 圆角: 4px
  - 背景: rgba(45,45,45,0.8)

右下角:
  - 比例尺
  - 坐标显示 (鼠标位置)
```

## 交互规范

### 地图操作

| 操作 | 行为 |
|------|------|
| 滚轮 | 缩放 |
| 拖拽 | 平移 |
| Shift+拖拽 | 框选缩放 |
| 右键拖拽 | 旋转/倾斜（可选） |
| 双击 | 缩放到点击位置 |

### 编辑操作

| 工具 | 操作 | 行为 |
|------|------|------|
| 选择 | 单击要素 | 选中并高亮 |
| 选择 | 再次单击 | 切换选中 |
| 选择 | 点击空白 | 取消选中 |
| 点绘制 | 单击 | 放置点 |
| 线绘制 | 单击 | 添加节点 |
| 线绘制 | 双击/回车 | 完成绘制 |
| 面绘制 | 单击 | 添加顶点 |
| 面绘制 | 双击/回车 | 闭合完成 |
| 节点编辑 | 拖拽节点 | 修改形状 |
| 删除 | 选中+Delete键 | 删除要素 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| Delete | 删除选中要素 |
| Esc | 取消当前绘制/取消选中 |
| V | 切换到选择工具 |
| P | 点绘制工具 |
| L | 线绘制工具 |
| G | 面绘制工具 (polyGon) |
