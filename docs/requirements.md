# 需求文档

## 项目概述

**项目名称**：地图编辑器 (Map Editor)
**项目类型**：纯前端 SPA 应用
**目标用户**：需要在地图上查看、编辑、导出空间数据的用户

---

## 功能需求

### F1 — 底图切换

| 项 | 说明 |
|------|------|
| 描述 | 用户可在必应地图和高德地图之间切换底图 |
| 默认 | 必应地图 |
| 底图列表 | 必应地图（Road/卫星）、高德地图（标准/卫星） |
| 切换方式 | 顶部工具栏下拉或按钮组 |
| 优先级 | P0 |

### F2 — 数据加载

| 项 | 说明 |
|------|------|
| 描述 | 用户可加载 GeoJSON 和 Shapefile 文件 |
| 支持格式 | .geojson, .json, .shp（需同时有 .dbf, .prj 等配套文件可选） |
| 加载方式 | 文件选择器拖拽或点击上传 |
| SHP 限制 | 纯前端解析，大文件（>50MB）性能受限，后续可演进 WASM |
| 坐标处理 | 自动识别源坐标系，转换为底图所在坐标系显示 |
| 优先级 | P0 |

### F3 — 多图层管理

| 项 | 说明 |
|------|------|
| 描述 | 支持同时加载多个数据图层 |
| 管理功能 | 图层可见性开关、图层顺序拖拽排序、图层重命名、删除图层 |
| UI 位置 | 左侧图层面板 |
| 编辑目标 | 一次激活一个图层用于编辑 |
| 优先级 | P0 |

### F4 — 基础编辑

| 项 | 说明 |
|------|------|
| 描述 | 在激活图层的要素上进行编辑操作 |
| 绘制工具 | 点标记、折线、多边形 |
| 编辑操作 | 拖拽节点修改形状、删除选中要素 |
| 属性编辑 | 右侧属性面板展示选中要素的属性并支持编辑 |
| 交互方式 | 点击地图选中要素（高亮），再次点击其他要素切换选中 |
| 优先级 | P0 |

### F5 — 撤销/重做 (Undo/Redo)

| 项 | 说明 |
|------|------|
| 描述 | 编辑操作支持撤销和重做 |
| 范围 | 绘制、修改、删除、属性编辑 |
| 深度 | 支持至少 50 步历史 |
| 快捷键 | Ctrl+Z / Ctrl+Shift+Z |
| 优先级 | P1 |

### F6 — 图片导出

| 项 | 说明 |
|------|------|
| 描述 | 截取当前地图视口保存为图片 |
| 格式 | PNG |
| 内容 | 包含底图 + 所有可见图层 |
| 分辨率 | 当前视口分辨率 |
| 触发方式 | 工具栏按钮，可配置文件名 |
| 优先级 | P1 |

### F7 — 数据导出

| 项 | 说明 |
|------|------|
| 描述 | 将当前图层数据导出为文件 |
| 支持格式 | GeoJSON (.geojson)、Shapefile (.shp + .dbf + .prj) |
| 导出范围 | 当前选中的图层 |
| SHP 导出 | 浏览器端生成，使用 JSZip 打包为 zip |
| 优先级 | P1 |

### F8 — 坐标系支持

| 项 | 说明 |
|------|------|
| 描述 | 支持常见坐标系的识别和转换 |
| 支持坐标系 | WGS84 (EPSG:4326)、GCJ-02、BD-09 |
| 转换时机 | 加载时自动检测并转换到显示坐标系 |
| 导出选项 | 可选择导出坐标系 |
| 优先级 | P1 |

---

## 非功能需求

| 编号 | 需求 | 说明 |
|------|------|------|
| NF1 | 性能 | 单个图层 10,000 要素以下流畅渲染 |
| NF2 | 兼容性 | Chrome 90+, Edge 90+, Firefox 90+ |
| NF3 | 响应式 | 桌面端优先，最小支持 1280×720 分辨率 |
| NF4 | 离线 | 纯前端，无需后端服务，可离线使用 |
| NF5 | 体积 | 首次加载 JS bundle < 2MB |

---

## 不在此版本范围

- 用户系统/登录
- 在线协作
- 空间分析（缓冲区、叠加分析等）
- 服务端瓦片发布
- 移动端适配
- 3D 地图
- 热力图等高级可视化
